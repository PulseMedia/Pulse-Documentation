# Decode

## Decode.`base64(str)`

> Returns: **string decoded from Base64 ASCII string**

| Param  | Type                | Description  |
| ------ | ------------------- | ------------ |
| str  | <code>string</code> | string to decode |

##### Example:
```javascript
var decodedString = Decode.base64("TXkgVGV4dA==");
Pulse.log(decodedString) //outputs: My Text
```
