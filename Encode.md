# Encode

## Encode.`base64(str)`

> Returns: **Base64 Encoded ASCII string**

| Param  | Type                | Description  |
| ------ | ------------------- | ------------ |
| str  | <code>string</code> | string to encode |

##### Example:
```javascript
var encodedString = Encode.base64("My Text");
Pulse.log(encodedString) //outputs: TXkgVGV4dA==
```
